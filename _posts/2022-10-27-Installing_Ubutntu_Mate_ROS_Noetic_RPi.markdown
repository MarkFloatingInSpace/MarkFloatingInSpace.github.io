# Installing Ubuntu Mate 20.04 for ROS Noetic on Raspberry Pi 4

First, install the Raspberry Pi Imager from https://www.raspberrypi.com/software/. Either via the installer on Windows or vie the command line on Linux:

    https://www.raspberrypi.com/software/

Next download the Ubuntu Mate 20.04 https://www.raspberrypi.com/software/ arm64 image from https://releases.ubuntu-mate.org/20.04/arm64/.

Burn the image onto a micro SD card using the Raspberry Pi Imager.

   sudo apt update
   sudo apt upgrade
